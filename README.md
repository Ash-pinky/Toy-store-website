A small **toy store website** built as a group assignment using **HTML, CSS, JavaScript, and PHP** with a MySQL backend.  
Features both **Customer** and **Admin** sides:

- **Admin**: Product CRUD (create/read/update/delete), member listing, order management.
- **Customer/User**: Browse and search toys, add to wishlist, add to cart and checkout (demo flow), view order history.

## Features (high level)

- Product listing with images and descriptions
- Admin CRUD for products (images, price, stock)
- Member registration & login
- Wishlist and shopping cart for users
- Order placement & admin order management
- Basic search and filtering
- Simple role-based access (admin vs user)

---

## Requirements

- PHP 7.4+ (or PHP 8.x)  
- MySQL  
- XAMPP (recommended) or any LAMP/WAMP stack  
- A modern browser (Chrome/Firefox/Edge)
